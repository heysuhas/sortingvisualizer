Documentation

How to Use

  1. Enter comma-separated numbers in the input field (e.g., 64,34,25,12,22,11,90).
  2. Click "Set Array" to initialize the visualization.
  3. Choose a sorting algorithm from the dropdown menu.
  4. Adjust the speed using the slider if desired.
  5. Click "Start Sorting" to begin the visualization.

Sorting Algorithms

Bubble Sort

Time Complexity: O(n²) in worst and average cases, O(n) in best case.

Description: Repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order.

Selection Sort

Time Complexity: O(n²) in all cases.

Description: Divides the input list into two parts: a sorted portion and an unsorted portion. Repeatedly selects the smallest element from the unsorted portion and adds it to the sorted portion.

Insertion Sort

Time Complexity: O(n²) in worst and average cases, O(n) in best case.

Description: Builds the final sorted array one item at a time. It iterates through the input elements, growing the sorted array with each iteration.

Merge Sort

Time Complexity: O(n log n) in all cases.

Description: Divides the unsorted list into n sublists, each containing one element, then repeatedly merges sublists to produce new sorted sublists until there is only one sublist remaining.

Quick Sort

Time Complexity: O(n log n) on average, O(n²) in worst case.

Description: Selects a 'pivot' element and partitions the other elements into two sub-arrays according to whether they are less than or greater than the pivot. The sub-arrays are then sorted recursively.
